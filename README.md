### Image Generator

This is full-stack application made with *Node* and *React*. Use prompts to generate an image from few words and ideas. With help of DALLE2 and OpenAI generating is easier than ever.


### How to run?


Install packages

```
npm i
```

Populate .env variables

```
PORT=
OPENAI_API_KEY=
NODE_ENV=
```

Run client

```
npm run dev
```

Run server

```
npm run dev
```
